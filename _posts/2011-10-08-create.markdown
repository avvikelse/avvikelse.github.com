Creates a new deviation.

### Example

#### Request

{% highlight bash %}
curl -X POST http://api.av.vikel.se/v1/deviations/ \
     -d "comment=Stopp vid Hornstull" \
     -d "scope=4" \
     -d "latitude=59.316331" \
     -d "longitude=18.034143" \
     -d "route_type=BUS" \
     -d "source=crowd"
{% endhighlight %}

#### Response

{% highlight javascript %}
{
    "id": "4e908ebd0f759d3c6e000000"
}
{% endhighlight %}
